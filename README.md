# polygonMission
ployMission for UAV
作者 qq 843230304
无人机多边形航线规划算法
1 任意多边形求取面积
2 任意多边形，按照喷幅规划航线
一般用于农业植保对农田作业区规划航线

# 2018 年 1月 12日 修改
1 增加 .qmake.conf 配置文件，配置输出目录 app
2 在 MainWindow 构造函数 里面 可以设置 自定义作业区域 多边形的顶点
3 运行算法后 app目录下 生成 polygonMissionFile.mission 文件可以用 qgroundcontrol 加载
